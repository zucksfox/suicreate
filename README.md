## Cara Install

Buka terminal kamu + Download dulu

```bash
  git clone https://github.com/zucksfox/suicreate.git
```
Buka Folder kamu
```
cd suicreate
```
Install library
```
pip insall -r requirements.txt
```
Run
```
python sui.py 
```
